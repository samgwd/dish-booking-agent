# Next.js starter

Minimal starter for a React + Next.js app using the App Router.

## Quick start

```bash
bun install
bun dev
```

Visit http://localhost:3000. All scripts work with `bun`:
- `bun dev`
- `bun build`
- `bun start`
- `bun lint`

## Features

- Chat UI at `/` with preset prompts and example conversation
- Booking form at `/booking` mirroring the original prototype
- Tailwind CSS configured with dark-mode support

## Scripts

- `npm run dev` – start the dev server
- `npm run build` – production build
- `npm run start` – run the production build
- `npm run lint` – lint with Next.js defaults
